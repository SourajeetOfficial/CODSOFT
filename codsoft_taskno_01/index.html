Projects

Gesture-Based Volume Control System

HCI project

This project demonstrates a basic Human-Computer Interaction (HCI) application that uses hand gestures to control the system's audio volume. By leveraging computer vision and hand-tracking technology, this project showcases how intuitive gestures can replace traditional input devices.

Camera Check: Ensures the system's camera can capture video frames. Provides a preview of the camera feed.

Hand Tracking Module: Detects hands and their landmarks in real-time using MediaPipe. Identifies hand gestures and calculates distances between landmarks.

Gesture-Based Volume Control: Adjusts system volume based on the distance between the thumb and index finger (confirmation gesture). Provides a visual representation of the current volume level on the screen.
